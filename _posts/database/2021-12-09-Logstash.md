> 服务端数据处理管道，能够从多个来源**采集**数据，**转换**数据，将数据**输出**到存储库(如：ES)。



# 安装



# 启动

* 指定配置文件

```shell
./bin/logstash -f logstash.conf
```

* 指定配置参数

```
./bin/logstash -e "input { stdin { } } output { stdout {} }"
```

终端输入命令，日志终端输出

# 结束

```shell
CTRL-D
```

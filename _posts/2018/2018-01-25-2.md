## php判断是否Json字符串 ##

代码:
```
 public function is_json($string) {
     json_decode($string);
     return (json_last_error() == JSON_ERROR_NONE);
}
```

json_last_error()函数返回数据编解码过程中发生的错误.

注意: json编解码所操作字符串必须是UTF8的.
